Пояснення: Зменшення font-size:

Шрифт .hero-title зменшено до 42px. Розрахунок нижнього padding:

В оригінальному коді для шрифту 52px використовується padding-bottom: 412px.
Якщо зменшити розмір шрифту на 19% (з 52px до 42px), нижній відступ пропорційно
зменшується приблизно на 19%. Результат: 412px - (412px \* 0.19) = 332px
